
# City Navigation System using Dijkstra’s Algorithm

## Overview
This project is a simple city navigation system that finds the shortest path between locations using Dijkstra’s Algorithm.

## Features
- Finds shortest route between two places
- Calculates total distance
- User-friendly input
- Graph visualization of route

## Technologies Used
- Python
- Heap Queue
- NetworkX
- Matplotlib

## How It Works
The city is represented as a weighted graph where:
- Nodes = locations
- Edges = roads
- Weights = distances

Dijkstra’s Algorithm is used to calculate the shortest path from source to destination.

## Example
Source: Home  
Destination: Station  
Output: Home -> Market -> Hospital -> Station  
Distance: 11 km

## Applications
- GPS navigation
- Delivery route optimization
- Emergency vehicle routing
- Smart city systems
